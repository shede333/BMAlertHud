# BMAlertHud

`create by shede333`


模仿系统的 **UIAlertView** ，但是 **UIAlertView** 不支持title 和 message 的文本格式自定义化；  
所以就仿照系统的 **UIAlertView** 原始功能，支持使用 **NSAttributedString** ，这样就可以自定义文本格式了，效果如下图;

###支持的功能：

* 支持叠加的弹出多个框；
* 适配ios6、7、8，
* 弹框支持随着屏幕的方向旋转而旋转
